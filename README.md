# Project over view
**This is My first ever project** was on data analysis using microsoft excel. The aim was to gather, sort, clean and analyse data for the purpose of gaining meaningful insights. In the course of the project, I have learned how to use Excel functions, formulas, sorting, filtering, pivot tables and charts to look for trends and patterns in the data.
I learned how to prepare data for cleaning, visualize it and report it in the project. This experience helped me understand the value of using data for decision-making and to address real-world issues. It also helped me to develop analytical thinking, attention to detail, and presentation skills.
This project was my first time getting into data analytics and was a good stepping stone for future data analysis projects.
